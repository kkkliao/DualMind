# DualMind Prompt Files

DualMind connects to OpenClaw and Hermes as local agent systems. It does not assume a fixed underlying model or provider.

- `oc-prompt.txt`: OpenClaw group-chat persona and collaboration rules.
- `hermes-prompt.txt`: Hermes group-chat persona and collaboration rules.

Model/provider selection belongs to OpenClaw and Hermes configuration, not DualMind.
